# Modules

Forage provides ready-to-use bean factories for a wide range of Apache Camel components.

## AI

- [Agents](ai/agents.md) — Composable AI agents with tool use
- [Chat Models](ai/chat-models.md) — OpenAI, Ollama, Gemini, Anthropic, and more
- [Embeddings](ai/embeddings.md) — Text embedding providers
- [Vector Databases](ai/vector-databases.md) — Qdrant, Milvus, PGVector, and more
- [Chat Memory](ai/chat-memory.md) — Message window, Infinispan, Redis

## Data

- [JDBC](jdbc.md) — Datasource providers with connection pooling
- [JMS](jms.md) — Connection factories for message brokers
- [Spring RabbitMQ](spring-rabbitmq.md) — CachingConnectionFactory for AMQP messaging

## Integration

- [CXF](cxf.md) — SOAP web service endpoints

## Security

- [TLS](security/tls.md) — SSLContextParameters from properties (keystore, truststore, cipher suites)
